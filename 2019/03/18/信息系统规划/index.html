信息系统规划

系统规划主要任务

  1. 确定信息系统及项目的优先顺序
  2. 组建信息系统项目团队
  3. 确定信息系统项目范围
  4. 启动项目

优先顺序

  • 判断开发系统能力的优先级

  • 指导委员会：来自IT部门和用户部门等高层管理者。

  • 项目开发小组: 项目经理、系统分析、设计、实施人员等组成，按照预期进度、质量和成本完成项目。

企业信息规划法(Business System Planning， 简称BSP法): 从企业目标入手，逐步将企业目标转化为管理信息系统的目标和结果，从而更好地支持企业目标的实现。

课程概述

信息分析与决策是本学期唯三的一门课程，也是专业里非常实用的一门课程，算是点亮一个SPSS和做调研问卷的技能树吧,如果以后想要从事数据方面的工作帮助也是很大的。单有技术不懂理论已经越来越走不动了。

信息源于分析，分析的过程是逐步认识事物，理解失误的过程，其目的就是发觉或发现有价值的知识或规律帮助人们

  • 描述 Description (What, 现状)
  • 解释 Explanation (Why, 影响因素)
  • 预测 Prediction (How, 建模，支持决策和行动)
  • 控制 Control 事务(Goal, 目的)
  • 最终使人类的生活状态更好

科学研究的特征：

  1. 系统性 Systematicness
  2. 实证性 Empirical character
  3. 客观性 Objectivity

总结：科学研究是一个连续，循环发展的过程，其成果具有累积性。当我们提出某个研究问题，从一个混沌位置的谜团，逐步理清脉络，建立一条解释性的知识架构到最后能够提出一套为人们所运用的使用知识，展现了科学活动的严谨和丰富的内容，为人类生活带来源源不绝的发展动力。

科学研究采取的是”有系统的实证研究方法”所进行的研究。

研究的基本元素：

  • 对象(Objective): 数字或数字矢量构成的事物
  • 对象的属性(Attributes): 属性的质或量上的变化，变量Variable
  • 假设(Hypothesis): 描述变量与变量的关系

Note： 这里的基本元素和编程的对应概念可以做类比

研究方法：

  • 描述性研究(Descriptive research): 汇报现象的各种描述性统计参数，研究对象的各种统计分析
  • 探索性研究(Exploratory research): 对各变量之间的关系进行描述
  • 验证性研究(Confirmatory research): 对某一现象进行假设，并用统计的方法进行验证

Note: 全部统计学和数理分析的方法啊…

描述性研究

基于可视化，用图形来描述数据特征、分布与关系

探索性研究

描述数据的分布，探索数据的状态及潜藏的规律。采用数值，图表等方法

验证性研究

常常采用回归分析的方法

  • 相关分析
  • 线性回归分析

研究工具：
统计(Statistics)是数据分析领域的重要工具，是收集，组织，分析，解释和表达数据的科学。基于统计的数据分析方法是本课程讲授的重点内容

在社会科学领域，特别是管理科学领域，常用的调查研究方法可以用以下流程示意:

  1. 研究目标
  2. 问卷设计
  3. 抽样调查、实验设计
  4. 采集、处理数据
  5. 分析数据
  6. 结论
  7. 辅助决策
  8. 实践检验

工具软件:

  • SPSS(课程使用，点亮技能树)
  • SAS(真实强大，然而暂时没有计划用)
  • R语言(未来计划)
  • Python(熟手工具和主力，课程外扩展，熟悉使用SciPy进行描述统计、参数估计和回归分析等操作)

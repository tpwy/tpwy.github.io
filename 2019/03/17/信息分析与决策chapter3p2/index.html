数据处理与数据核查

这节开始就要接触到两个在学习Python数据分析中的老朋友遗漏值处理和箱型图了，18年美赛C题数据分析题的那个数据至今让我印象深刻，遗漏的太夸张了导致自己不得不大批大批的删特征，剩下的再用平均数或者缺省值补上，可惜那次还没能好好利用方差和箱型图再多搞掉一些比较极端的数据和特征。

3.2 数据的核查与整备

  • 过程核查：在数据录入过程中进行
    • 目的： 维持数据输入过程的正确性
    • 定点查核：根据数据输入工作的流程，在适当的段落进行数据检查
    • 定时查核：以时间为单位，在特定时间段进行数据的检查
    • 专人核查：有经验的研究人也
  • 终点核查：在数据完成输入之后进行
    • 模式1：可能性核查：核查内容包括数据格式的确认、是否有重复录入数据、数据中是否有超过范围数值(out-of-range value)。核查方法可以利用描述统计的相关技术
    • Case1: 查找重复项：SPSS功能 数据/标识重复个案 可以按照设定的依据寻找是否存在重复的个案。再利用 数据/选择个案 来删除重复个案。
    • 模式2: 逻辑性核查(Logical code checking)：核查内容为问题中答案逻辑是否合理。核查方法同样是利用描述统计工具或进行前期的探索性研究。
    • Case2: 逻辑检验：采用定制表的方式来查看是否存在逻辑问题，使用SPSS功能 分析/表/定制表 ，可以按照自己的想法组合不同的特征并生成表格。

数据核查：确保数据输入过程的正确无误，是侦错的过程，经过这一过程的数据为Computerized raw data,其数据格式，内容与排列方式，均与文本资料完成一直，并符合编码系统的编码原则。

遗漏值处理(Missing data)

遗漏值处理是量化研究中，最容易出现且必定干扰结构分析的问题，其最大的影响是造成样本的流失

  • 可忽略遗漏(Ignorable missingness)：随机性的导致，通常的影响只是样本数量多少的问题，可直接删除，或采用不同的补救方法填补，对统计分析的影响视为一种随机变异的来源，影响不大。
  • 不可忽略的遗漏(non-ignorable missingness):对于研究结果与分析过程影响较大，任意填补或估计会造成一致性的高估或低估，甚至于遗漏笨晒可以作为研究的解释变量，称为讯息性遗漏(Informative drop-out)

处理方法：

  • 事前预防法：增加”其他”选项，采用开放式问题方式
  • 删除法：
    • 完全删除：处理时，任何一个变量出现遗漏，就将与该受测者有关的数据整笔删除
    • 配对删除：分析时牵扯的变量有遗漏时才加以排除。分析前并不删除，分析指令下达后，才将不完整的信息过滤掉。
    • 两种删除法分别对应SPSS中的全列删除(Exclude cases listwise)和配对删除(Exclude cases pairwise)。可以在相关分析和因子分析中找到这两个选项。
  • 取代法: 运用推理补漏，比较常用的方法有取中间数和平均数。
  • 回归估计法：运用统计回归原理，预测遗漏变量中的可能取值，常用于时序分析。

在分析之前，可以使用置换遗漏值功能来处理遗漏值(Transform/Replace Missing Values):

  • Series means: 表示用整个序列的均值作为替代
  • Mean of nearby points: 表示利用邻近点的均值替代
  • Median of nearby points: 表示利用邻近点的中位数作为替代(nearby 方法对于前后同样有遗漏值的数据无效)
  • Linear interpolation: 线性差值法，用缺失值前后两点数据的某种线性组合进行填补，是一种加权平均
  • Linear trend at point: 线性趋势法，表示利用回归拟合线的拟合值作为替代

一般情况下，当样本观察值在少数几个题项上有遗漏时，将样本问卷作为无效问卷会浪费许多有用的信息，此时可以采用置换缺失值的方法对缺失值进行补漏

若一个样本观察值在问卷上有多项缺失的情况，最好作为无效卷处理，而不要采用补漏方法，因为这样有可能会使统计分析结果有偏差。

偏离值的侦测与处置

偏离值(Outliers): 是指变量中偏离正态，不寻常的数值，就是与多数受测者的反应数值极端不同的情况。

极端值(Extremes): 严重偏离的情况

Case3: SPSS的预检功能： 简单的描述统计和图表法

  1. 单变量偏离检测,兼具差错的功能: 分析/描述统计/描述 选项里，可以获得描述统计值，通过观察判断最大最小值是否超出五点量表的设置，探知可能出现的偏离值或错误
  2. 利用图表观察： 分析/描述统计/频率 中，可以生成带正态分布的直方图，根据图表的显示结果判断是否存在极端值或偏离值。
  3. 如果要完整地了解数据的分布特性，观察异常值，箱线图(BoxPlots)可以说是一种非常好的图示方法

箱线图中

  • 距离盒长的1.5倍以上被认为是偏离值
  • 距离盒长的3倍以上被认为为极端值
  • 极端值与偏离值越多，代表偏离情形越严重
  • 箱线图可以在 分析/描述统计/探索 中找到使用选项，同时还可以通过设置因子列表来对样本进行分类，再观察其分类后的分布情况。

其他的一些重要数据处理功能

数据转换：

  1. 生成新变量: 转换/计算变量。能够基于 已有的变量生成模型需要的新变量
  2. 数据分组: 转换/可视分箱。 可以将连续型变量转换成分类变量

总结：

  1. 利用SPSS的数据栏目中提供的标识重复个案功能查找重复项
  2. 使用定制表检查数据存在的逻辑问题
  3. 根据实际情况选择合适的方法来处理遗漏值，学会使用转换/替换缺失值功能，以及了解分析过程中全列删除和配对删除的意思
  4. 使用描述系统、直方图、箱线图来侦测和处置偏离值

SPSS内置的功能还是非常非常丰富的了，基本上依靠点点点就可以实现很多复杂的操作，在自定义方面也提供了足够多的选项和设置供使用者挑选。不过还是想吐槽吐槽这个诡异蓝的UI界面以及糟糕的中文翻译，怎么这些功能名称看上去这么怪别扭的，还不如直接用英文原版的界面。另外一点就是提供的功能实在是太多了以至于有时候想要使用某个功能要点点点找半天才能找到，但是这个可能是因为我用得少的缘故哈哈。
